Detailed Description

The unit test for compilation.

Author
Autin (SV-Engineer)
This file contains the unit test used to ensure the SDL2 libraries can be compiled.
To compile:
  1. Change directory to the top level (probably TULIP), but if not, whatever directory contains the makefile
  2. Use command: > mingw32-make makefile all UT=compilation
